# Реализованные тесты:





1. Проверка условий длины имен для книг
2. Проверка на тип данных для имен книг
3. Проверка на отсутствие в словаре жанра книги при первичном добавлении книги
4. Проверка добавления последовательно двух книг
5. Проверка установки жанра при положительных условиях наличия книги в словаре и жанра в init списке
6. Проверка неустановки жанра при отсутствии книги в словаре
7. Проверка неустановки жанра при отсутствии жанра в init списке
8. Проверка получения жанра книги по имени
9. Проверка получения списка книг по жанру при наличии жанра в init списке
10. Проверка получения списка книг по жанру при отсутствии жанра в init списке
11. Проверка получения типа данных словаря
12. Проверка на исключение из словаря для детской аудитории недопустимых жанров книг
13. Проверка недобавления в избранное книги, которой нет в словаре
14. Проверка недобавления в избранное книги, которая уже содержится в избранном
15. Проверка добавления в избранное книги, которая есть в словаре и не содержится в избранном
16. Проверка неудаления из избранного книги, которой нет в избранном
17. Проверка удаления книги из избранного 
18. Проверка получения списка избранного
19. Проверка типа данных списка избранного
